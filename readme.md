# THe Ghost Engine

![demo](ghost-engine-demo.gif)

## A 2D top-down shooter where you pilot a ship and are assisted by ghostly allies. Fight waves of enemies, which by then became your allies manage energy, and upgrade your arsenal to survive as long as possible.

### Controls (Will Change in future updates)
- Move: Arrow keys or WASD
- Shoot -> H for primary weapon, J for secondary, K for tertiary, K for quaternary weapons
- Summon Ghosts: Spacebar (costs energy)
- Parry Missiles: X (short window, costs energy)
- Cancel Summon: returns all ghosts to available queue
- Change Formation: 1-4 for Line, Circle, V-shape
- Single Summon: F keys, F1-F4 for specific ghost types
