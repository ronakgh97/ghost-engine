# THe Ghost Engine

![demo](ghost-engine-demo.gif)

## A 2D top-down shooter where you pilot a ship and are assisted by ghostly allies. Fight waves of enemies, which by then became your allies, consume your energy while deployed.

### Controls (Will Change!)
- Move: Arrow keys or WASD
- Shoot -> H for primary weapon, J for secondary, K for tertiary, K for quaternary weapons
- Summon Ghosts: Space-bar (costs energy)
- Parry Missiles: X (short window, costs energy)
- Cancel Summon: C (returns all ghosts to available queue)
- Change Formation: 1-3 for Line, Circle, V-shape (Needs Optimal Ghosts in queue)
- Single Summon: F keys, F1-F6 for specific ghost types (Basic, Sniper, Tank, Elite, Healer, Splitter)
